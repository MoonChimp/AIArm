## **ğŸ¯ API Endpoints**

### **Core Chat & AI**
- `POST /api/chat` - Conversational AI with memory
- `GET /api/status` - System status and health
- `POST /api/personality/mode` - Set AI personality
- `POST /api/personality/honesty` - Set honesty level

### **Image & Media Generation**
- `POST /api/cinema/generate` - Generate images with SDXL
- `POST /api/cinema/storyboard` - Create storyboards
- `POST /api/cinema/video` - Generate video sequences
- `GET /api/cinema/image/<filename>` - Serve generated images

### **Memory & Learning**
- `GET /api/memory/sessions` - List conversation sessions
- `GET /api/memory/context/<session>` - Get session context
- `POST /api/learning/record-correction` - Record user corrections
- `GET /api/self-awareness` - Get self-awareness data

### **Machine Learning Integration**
- `POST /api/ml/dump-memory` - Export conversations for training
- `POST /api/ml/dump-knowledge` - Export learned knowledge
- `POST /api/ml/fine-tune` - Fine-tune models
- `GET /api/ml/models` - List registered models

### **Code Execution & Files**
- `POST /api/execute` - Execute code/commands
- `POST /api/files/read` - Read files
- `POST /api/files/write` - Write files
- `POST /api/files/search` - Search files
