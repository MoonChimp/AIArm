WHAT ARE THEY?
================================================================

LIGHTWARE (Conservative Learning)
----------------------------------
Philosophy: "I don't know how, so I'll LEARN"

When Nexus fails at a task:
1. Analyze what skill is missing
2. Search for information/examples
3. Generate implementation
4. Store in skill library
5. Retry the task

Characteristics:
✓ Learns from existing knowledge
✓ Searches and studies
✓ Builds skill library over time
✓ Safe, gradual growth

DARKWARE (Aggressive Creation)
-------------------------------
Philosophy: "I don't know how, so I'll INVENT it"

When Nexus fails at a task:
1. Analyze what's needed
2. Create the solution from scratch
3. Install required dependencies
4. Integrate into Nexus (self-modification)
5. Execute immediately

Characteristics:
✓ Creates new tools/software
✓ Installs libraries automatically
✓ Self-modifies Nexus code
✓ Aggressive, autonomous growth

SECURITY CONSIDERATIONS
================================================================

LIGHTWARE SAFETY:
- Only creates Python code
- Stores in isolated directory
- Doesn't auto-install software
- Relatively safe

DARKWARE WARNINGS:
⚠ Automatically installs libraries
⚠ Self-modifies Nexus code
⚠ Executes created code immediately
⚠ VERY powerful, use with caution

Recommendation:
- Start with LightWare for most tasks
- Use DarkWare when you want aggressive growth
- Review created skills periodically

Enable/Disable:

darkware.disable_aggressive_mode()  # Safety mode
